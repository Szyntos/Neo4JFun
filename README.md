# Neo4j Graph Explorer

Simple tool to learn **Cypher** with a visual graph and exercises. 

---

## How to run

```bash
docker compose up --build
```

Then open:

```
neo4j-graph.html
```

---

## How it works

* **Left** → exercises
* **Center** → graph
* **Bottom** → write queries
* **Right** → filters & tools

Run query:

* Click ▶
* or `Ctrl + Enter`
